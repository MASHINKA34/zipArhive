# Quick Zip

## Описание

Quick Zip - настольное приложение для Windows, которое собирает выбранные
файлы и папки в один ZIP-архив. Источник выбирается через диалог, где можно
заходить в подпапки и отмечать галочками нужные файлы и каталоги, либо
перетаскиванием прямо в окно. Перед упаковкой видно количество выбранных
файлов и их суммарный размер. Имя архива по умолчанию подставляется из
названия выбранной папки.

Возможности:

- выбор папки и отметка нужных файлов и подпапок галочками в дереве;
- навигация по вложенным папкам прямо в окне выбора;
- перетаскивание файлов и папок в окно;
- отметка файлов галочками в списке, выбор всех сразу;
- разные цветные значки для разных типов файлов;
- упаковка в реальный ZIP-архив с индикатором прогресса;
- автоматическое имя архива по названию выбранной папки;
- тёмная тема оформления.

## Требования

- Windows 10 или 11
- Python 3.11 или новее (проверено на 3.12)
- PyQt6

Все зависимости перечислены в файле requirements.txt.

## Установка

Рекомендуется использовать виртуальное окружение.

    python -m venv .venv
    .venv\Scripts\activate
    pip install -r requirements.txt

Без виртуального окружения достаточно одной команды:

    pip install -r requirements.txt

## Запуск

Из корня проекта:

    python main.py

## Сборка в exe

Сборка одного исполняемого файла выполняется через PyInstaller.
Готовый скрипт уже настроен, достаточно запустить:

    build.bat

После сборки исполняемый файл появится в папке dist:

    dist\QuickZip.exe

Если в папке assets\icons лежит файл app.ico, он будет использован
как значок приложения. Если файла нет, сборка проходит без значка.

Команда, которую выполняет build.bat:

    pyinstaller --onefile --windowed --name QuickZip main.py
